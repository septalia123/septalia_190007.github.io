    Turunan Numerik¶

    Materi¶

    Persoalan adalah menentukan hampiran nilai turunan pada fungsi f dalam bentuk tabel.

    Turunan Numerik dapat dihitung dengan menggunakan 3 metode pendekatan, yaitu beda maju, beda mundur, dan beda pusat. Namun rumus yang di gunakan berbeda antara rumus dari turunan 1 dan rumus dari turunan 2. Perlu diperhatikan bahwa jarak antar titik yang dihitung harus sama.

    Turunan Pertama¶

    a. Metode Hampir Beda Maju¶

    ​ Rumus turunannya yaitu:

    \[ \begin{aligned}\\ f'\left( x_{0}\right) =\dfrac{f\left( x_{0}+h\right) -f\left( x_{0}\right) }{(x_{1}-x_{0})}\end{aligned} \]

    ​ ket: h adalah selisih antara 2 buat titik terdekat (x1-x0)

    b. Metode Beda Mundur¶

    ​ Rumusnya yaitu:

    ​

    \[ \begin{aligned}\\ f'\left( x_{0}\right) =\dfrac{f\left( x_{0}+h\right) -f\left( x_{-1}\right) }{(x_{0}-x_{-1})}\end{aligned} \]
    c. Metode Beda Pusat¶

    Rumusnya yaitu:

    \[ \begin{aligned}\\ f'\left( x_{0}\right) =\dfrac{f\left( x_{+1}\right) -f\left( x_{-1}\right) }{(x_{+1}-x_{-1})}\end{aligned} \]
    ¶

    Turunan Kedua¶

    ​ Turunan kedua yaitu bantuan deret Taylor.

    a. Metode Beda Maju¶
    \[ \begin{aligned}\\ f'\left( x_{0}\right) =\dfrac{(f\left( x_{+2})+2f(x_{+1})+f(x_{0})\right)}{(x_{+1}-x_{0})^2}\end{aligned} \]
    ¶
    b.Metode Beda Mundur¶
    \[ \begin{aligned}\\ f'\left( x_{0}\right) =\dfrac{(f\left( x_{-2})+2f(x_{-1})+f(x_{0})\right)}{(x_{-1}-x_{0})^2}\end{aligned} \]
    ¶
    c. Metode Beda Pusat¶
    \[ \begin{aligned}\\ f'\left( x_{0}\right) =\dfrac{(f\left( x_{+1})+2f(x_{0})+f(x_{-1})\right)}{(x_{0}-x_{-1})^2}\end{aligned} \]

    Algoritma¶

    Algoritma penyelesaian turunan numerik dengan menggunakan rumus polinom interpolasi:

    1. Menginisialisasikan data list dari x dan y

    2. Menginisialisasikan nilai x yang di cari yaitu xCari = 7

    3. Membuat List kosong untuk menampung data

    4. menampilkan data y dari list ke dalam bentuk matrik

    5. membuat tabel selisih terbagi

    6. menampilkan nilai dari tiap suku dan menampilkan hasil interpolasi

    Code Program¶

    #data x
x = [1, 2, 3, 4, 5, 6]
y = [14.5, 19.5, 30.5, 53.5, 94.5, 159.5]

xCari = 7

#membuat list kosong
ST = []
for i in range(len(x)):
  datakolom = []
  for j in range(len(x)-i):
    datakolom.append(0)
  ST.append(datakolom)

print(ST)

n = len(x)

#membuat y atauf(x) ke matrix
for k in range(0, n):
  ST[k][0] = y[k]

print ("matrix : ", ST)

#membuat Table Selisih Terbagi
for k in range(1, n):
  print("ST - {} : ".format(k))
  for i in range(0, n-k):
    ST[i][k] = (ST[i+1][k-1] - ST[i][k-1]) / (x[i+k] - x[i])
    print(ST[i][k])

print('matrix : ', ST)

jumlah = ST[0][0]
for i in range(1, n):
  suku = ST[0][i]
  for k in range(0,i):
    # print(x[k])
    suku = suku*(xCari-x[k])
  print("suku {} : ".format(i), suku)

  jumlah = jumlah + suku

print("hasil interpolasi dari x = {} adalah f({}) = ".format(xCari, xCari), jumlah)
